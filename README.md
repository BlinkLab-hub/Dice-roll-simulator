# Dice Roll Simulator

A simple JavaScript dice rolling app. Click the button to generate a random number between 1 and 6.

---

## Features

- Random dice roll (1–6)
- Button click interaction
- Simple UI using HTML, CSS, and JavaScript

---

## How It Works

Click the **Roll** button and a random number is displayed as the dice value.

```js
Math.floor(Math.random() * 6) + 1

```js
